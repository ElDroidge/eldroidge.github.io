This is Il1

Ok, keep in mind that these are all open source fonts, so they’re the ones that are freely available on Google Fonts. Otherwise it’s like trying to find a needle in a haystack if you’re also looking for licensed fonts.

So to begin with, there are some sans-serif fonts, like Advent Pro, Andika, Convergence, Exo, and Geo, that take too many liberties in my opinion, so I don’t like them because I can’t take them seriously lol.
